WebFinger¶

GET /.well-known/webfinger¶

Get account links

Example request:

GET /.well-known/webfinger?resource=acct:Sam@example.com HTTP/1.1
Content-Type: application/json

Example response:

HTTP/1.1 200 OK
Content-Type: application/jrd+json; charset=utf-8

{
  "subject": "acct:Sam@example.com",
  "links": [
    {
      "href": "https://example.com/user/Sam",
      "rel": "http://webfinger.net/rel/profile-page",
      "type": "text/html"
    },
    {
      "href": "https://example.com/federation/user/Sam",
      "rel": "self",
      "type": "application/activity+json"
    }
  ]
}
Query Parameters

  • acct (string) – user account

Status Codes

  • 200 OK – success

  • 400 Bad Request –

    • Missing resource in request args.

    • Invalid resource.

  • 403 Forbidden – Error. Federation is disabled for this instance.

  • 404 Not Found – user does not exist
