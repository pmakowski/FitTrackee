NodeInfo¶

GET /.well-known/nodeinfo¶

Get node info links

Example request:

GET /.well-known/nodeinfo HTTP/1.1
Content-Type: application/json

Example response:

HTTP/1.1 200 OK
Content-Type: application/json; charset=utf-8

{
  "links": [
    {
      "rel": "http://nodeinfo.diaspora.software/ns/schema/2.0",
      "href": "https://example.com/nodeinfo/2.0"
    }
  ]
}
Status Codes

  • 200 OK – success

  • 403 Forbidden – Error. Federation is disabled for this instance.

GET /nodeinfo/2.0¶

Get node infos

Example request:

GET /nodeinfo/2.0 HTTP/1.1
Content-Type: application/json

Example response:

HTTP/1.1 200 OK
Content-Type: application/json; charset=utf-8

{
  "version": "2.0",
  "software": {
    "name": "fittrackee",
    "version": "0.4.8"
  },
  "protocols": [
    "activitypub"
  ],
  "usage": {
    "users": {
      "total": 10
    },
    "localWorkouts": 35
  },
  "openRegistrations": true
}
Status Codes

  • 200 OK – success

  • 403 Forbidden – Error. Federation is disabled for this instance.
